## What's New

### Command
The **leap** command lets you quickly create new projects. It also watches for changes in your project and recompiles
the project when you have made changes.

### Web Server
The **web** module provides a lighting fast web server that is correct and complete.

The **route** module defines the rules which filter incoming requests from web clients.

### Data
The **data** module makes it easy to store and retrieve data from a sources such as file systems, key-value stores, 
databases and messaging brokers.

### Templating
The **view** module allows to create html reusable templates.
